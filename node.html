node.js

Install Node.js from source

Add users

sudo groupadd -r node

sudo useradd -M -r --shell /sbin/nologin --comment 'User for running node.js' -g node  --home /var/lib/node node

compile Node.js

wget http://nodejs.org/dist/v0.6.10/node-v0.6.10.tar.gz

tar -zxvf node-v0.6.10.tar.gz

cd node-v0.6.10

sudo ./configure

sudo make

sudo make install

cd ..

sudo rm -rf node-v0.6.10*
