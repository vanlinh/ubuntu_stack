MongoDB

Installing MongoDB using package manager

Add GPG key

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv 7F0CEB10

Open package manager's source list

sudo vi /etc/apt/sources.list

add

deb http://downloads-distro.mongodb.org/repo/ubuntu-upstart dist 10gen

save, close and run following commands

sudo apt-get update

sudo apt-get install mongodb-10gen

Configure MongoDB

Open MongoDB configuration file

sudo vi  /etc/mongodb.conf

and edit

dbpath = /data/db/mongodb
logpath = /data/db/mongodb.log
logappend = true

bind_ip = 127.0.0.1
port = 27017
fork = true

auth = true
nohttpinterface = true

Optional Open MongoDB Port (27017) on Iptables Firewall

sudo iptables -I INPUT -p tcp -m state --state NEW,ESTABLISHED --dport 27017 -j ACCEPT
